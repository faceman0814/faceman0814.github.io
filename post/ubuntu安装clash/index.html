Ubuntu安装Clash

faceman

十一月 9, 2024

  1. 下载配置文件或者导入订阅连接配置。

    1. 下载连接（由于clash作者删库跑路了，所以我这里提供一个安装包）

    链接：https://pan.baidu.com/s/1Rz5Sb-1hhCSy_XjgzpKWvA 提取码：vaaz 2. 解压,已解压可跳过

    gzip -d clash.gz

    1. 赋值权限

      sudo chmod +777 clash

    2. 第一次执行创建必要文件：./clash

      1. 安装过程可能会出现网络问题下载不了，可以直接复制下载链接到外部现在再传进去是一样的。

    3. 去代理软件复制节点订阅地址

      1. 这里可以复制所有节点地址也可以自己配置节点然后生成地址

    4. 配置文件获取。

      方法一

      命令行获取

      wget -U "Mozilla/6.0" -O ~/.config/clash/config.yaml 你的Clash订阅链接网址
## 主订阅链接：https://wmdy.355365.xyz/link/w1FRr398wFCOa0XK?clash=2
## 备用订阅链接：https://dy.wmyun.men/link/w1FRr398wFCOa0XK?clash=2

      方法二

      1. 用windows的clash软件获取配置文件
      2. 将获取的config文件替换到/root/.config/clash/config.yaml

    5. 最后的文件结构应该是这样

    6. 代理改成手动并配置端口，如果是云服务器，需要打开对应的端口。

      # 编辑环境变量
sudo nano /etc/environment
# 配置手动代理
http_proxy="http://127.0.0.1:7890"
https_proxy="https://127.0.0.1:7890"
socks_proxy="socks5h://127.0.0.1:7891"
no_proxy="localhost,127.0.0.1,::1"
# CTRL+X 保存

# 应用环境变量 
source /etc/environment
# 查看代理服务配置
env | grep -i proxy
# 重启
sudo reboot

    7. 配置服务

      1. vim /etc/systemd/system/clash.service

        [Unit]
Description=Clash - A rule-based tunnel in Go
Documentation=https://github.com/Dreamacro/clash/wiki
[Service]
OOMScoreAdjust=-1000
ExecStart=/usr/local/bin/clash -f /root/.config/clash/config.yaml
Restart=on-failure
RestartSec=5
[Install]
WantedBy=multi-user.target

      2. 将 clash执行程序拷贝到 /usr/local/bin

        cp clash /usr/local/bin

    8. 启动Clash

      # 执行
sudo systemctl enable clash
sudo systemctl start clash
sudo systemctl status clash
journalctl -xe

# 重启clash服务
sudo systemctl restart clash
# 查看clash服务状态
sudo systemctl status clash
# 启动clash服务
sudo systemctl start clash
# 停止clash服务
sudo systemctl stop clash
# 设为开机启动
systemctl enable clash

    9. 在官方提供的配置链接里指定节点

      1. 打开链接

        http://clash.razord.top/

      2. 配置

    12. 想恢复正常上网取消代理或者把配置的端口删除然后重启即可。

      # 取消代理

# unset http_proxy
# unset https_proxy

docker compose安装clash

参考链接：ubuntu-server20.04安装clash，并快速配置到可用 - linux学习记录 (frank9.com) docker-compose.yml

version: '3.9'
services:
  clash:
    image: dreamacro/clash
    container_name: clash
    volumes:
      - ./.config/clash:/root/.config/clash
    # - ./ui:/ui # dashboard volume
    ports:
      - "7890:7890"
      - "7891:7891"
      # - "8080:8080" # external controller (Restful API)
      # TUN
      # cap_add:
      # - NET_ADMIN
      # devices:
      # - /dev/net/tun
    restart: unless-stopped
    network_mode: "host" # or "host" on Linux
	```
